# hack-answered
Repository stores the hackathon App QAnserd
   
=======  

## How to run?  
  
  
1. Install dependencies  
`pip install -r requirements.txt`  
2. Start flask server   
`python app.py`  
3. Go to browser, url:  
`localhost:5000/`  
   
   
========   
  
## Note:

1. Use Python 3.
